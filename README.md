# Blackjack Trainer

A browser game to practice basic strategy and card counting (Hi-Lo).

This is the practice tool for students in the Atlas Research - Card Counting and Quantitative Finance II course. A way to get comfortable with the basics of Blackjack, probability, expected value, and decision-making under uncertainty, which is the same stuff that shows up later in quant finance.

## What it does

- Practice your blackjack skills against a computer dealer (4, 6, or 8 decks)
- Basic Strategy mode: tells you if your hit/stand/double/split/surrender call was correct
- Card Counting mode: shows a live running count and true count
- Wallet Function: practice with a €10,000 starting bankroll
- Hand history and performance chart
- A few color themes
